## Code style
- Use module paths that preserve namespaces, e.g path::PathBuf instead of just PathBuf.
- Adhere to the Rust API-Guidelines: https://rust-lang.github.io/api-guidelines/about.html

## Commit messages
- Adhere to the Conventional Commits specification: https://www.conventionalcommits.org/en/v1.0.0/
- Always specify type.
- Specify scope where applicable.
